# Student-Result-Portal
A student result portal is a portal where the students can view their results and admins can register students and update marks in the portal.
Developed using Java, Java Swing, SQL, and JDBC, resulting in a 30% reduction in data entry time. Admins can manage 500+ studentsâ€™ data and results, add students and their results, and students can search their results by roll number, resulting in a 40% increase in student satisfaction.
